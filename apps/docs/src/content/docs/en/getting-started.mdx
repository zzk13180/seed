## Prerequisites

Before starting, ensure your development environment meets these requirements:

| Tool        | Version      | Notes                                  |
| ----------- | ------------ | -------------------------------------- |
| **Node.js** | `>= 22.x`    | LTS version recommended                |
| **pnpm**    | `>= 10.10.0` | Efficient package manager              |
| **Rust**    | `>= 1.75`    | For Desktop app development (optional) |
| **Docker**  | `>= 24.x`    | For local database (optional)          |

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-org/seed.git
cd seed
```

### 2. Install Dependencies

```bash
# Install all dependencies using pnpm
pnpm install
```

### 3. Configure Environment

```bash
# Copy the environment template
cp apps/server/.env.example apps/server/.env

# Edit the config file with your settings
# e.g., database connection, JWT secret, etc.
```

### 4. Start Development Server

```bash
# Start Admin console
nx dev @seed/admin

# Start Server API (in another terminal)
nx dev @seed/server
```

🎉 Visit [http://localhost:5173](http://localhost:5173) to see the Admin console!

## Common Commands

```bash
# Nx commands (recommended)
nx dev @seed/admin          # Develop admin
nx dev @seed/server         # Develop server
nx build @seed/admin        # Build single project
nx run-many -t build        # Build all projects
nx graph                    # View dependency graph

# Testing
pnpm test                   # Run all tests
nx test @seed/server        # Run single project tests

# Code Quality
pnpm lint                   # ESLint + Stylelint + Prettier
pnpm type-check             # Full type check

# Database Management
cd apps/server
pnpm db:generate            # Generate migrations
pnpm db:migrate             # Run migrations
pnpm db:studio              # Drizzle Studio
pnpm docker:db              # Start local PostgreSQL
```
