## 欢迎使用 Seed 🌱

Seed 是一个**企业级全栈多平台开发模板**，为现代应用开发提供生产就绪的架构蓝图。

### 为什么选择 Seed？

| 特性            | 描述                                    |
| --------------- | --------------------------------------- |
| 🔌 **框架无关** | Controller 模式将业务逻辑与 UI 框架解耦 |
| 📝 **类型安全** | 前后端共享类型定义，编译时捕获错误      |
| ⚡ **极速开发** | Nx 缓存加速，智能增量构建               |
| 🌐 **多平台**   | Web、Mobile、Desktop 一套代码           |

### 技术栈概览

```
┌─────────────────────────────────────────────────────────────┐
│                        用户交互层                           │
├──────────────┬──────────────┬──────────────┬───────────────┤
│  @seed/admin │ @seed/mobile │ @seed/desktop│  @seed/docs   │
│   Web 控制台 │   移动巡检   │  桌面监控    │   产品文档    │
└──────┬───────┴──────┬───────┴──────┬───────┴───────────────┘
       │              │              │
       └──────────────┼──────────────┘
                      │ HTTP/WebSocket
              ┌───────▼───────┐
              │ @seed/server  │  Hono + Bun API
              │ 认证/授权/更新 │
              └───────┬───────┘
                      │ rosbridge / 原生协议
              ┌───────▼───────┐
              │ @seed/robotics│  ROS 机器人核心
              │ 导航/建图/控制 │
              └───────────────┘
```
