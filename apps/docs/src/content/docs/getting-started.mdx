## 环境要求

| 工具    | 版本   | 说明                    |
| ------- | ------ | ----------------------- |
| Node.js | ≥ 22   | LTS                     |
| pnpm    | ≥ 10   | 包管理                  |
| Rust    | ≥ 1.75 | 桌面端开发（可选）      |
| Docker  | ≥ 24   | server 本地开发（可选） |

## 启动

```bash
git clone https://github.com/zzk13180/seed.git
cd seed
pnpm install

# 启动管理后台
pnpm nx dev @seed/spa-admin

# 启动 Edge API 服务
pnpm nx dev @seed/api-edge

# 启动 Bun Docker 后端
pnpm nx dev @seed/api-bun
```

访问 `http://localhost:5173` 查看 admin。

## 常用命令

```bash
pnpm nx dev <project>           # 开发模式
pnpm nx build <project>         # 构建单个项目
pnpm nx run-many -t build       # 构建全部
pnpm nx run-many -t test        # 运行测试
pnpm nx run-many -t lint        # Lint 全部
pnpm nx graph                   # 可视化依赖图
```
