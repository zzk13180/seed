## <DocIcon name="lucide:clipboard-list" /> 环境要求

在开始之前，请确保您的开发环境满足以下要求：

| 工具 | 最低版本 | 推荐版本 |
|------|---------|---------|
| **Node.js** | 18.x | 22.x |
| **pnpm** | 9.x | 10.x |

## <DocIcon name="lucide:rocket" /> 安装

### 1. 克隆仓库

```bash
git clone https://github.com/your-org/your-repo.git my-project
cd my-project
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 启动开发环境

```bash
# 启动文档开发服务器
pnpm --filter @seed/docs dev
```

## <DocIcon name="lucide:package" /> 构建

```bash
# 构建生产版本
pnpm --filter @seed/docs build
```

## <DocIcon name="lucide:wrench" /> 自定义

- 编辑 `src/config.ts` 配置站点设置
- 在 `src/pages/` 目录添加新页面
- 在 `src/assets/styles/` 自定义样式
